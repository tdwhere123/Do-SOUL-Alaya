CLEARED. v0.3.1 closed as a patch-safe internal quality release on
2026-05-13. The patch improves recall and MCP memory handler
maintainability, records a targeted recall baseline, and bumps workspace
packages `0.3.0` -> `0.3.1`.

- Core recall candidate construction moved into
  `packages/core/src/recall-candidate-builder.ts`.

  `RecallService.recall()` and `RecallResult`.

  `mcp-memory-tool-handler.ts` to
  `mcp-memory-recall-result.ts`; tool catalog and response schemas are
  unchanged.

  `fixtures:host-autonomy-witness`, and root `better-sqlite3` is declared
