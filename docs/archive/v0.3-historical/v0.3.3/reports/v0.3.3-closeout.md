## Status

CLEARED on `work/v0.3.x-quality-recall` before merge. v0.3.3 is a
patch-internal quality release for recall graph behavior, bootstrap

## Scope Compliance

- Public MCP tool names, descriptions, request schemas, and response
  schemas are unchanged.
- Protocol zod schemas, EventLog payload schemas, runtime config schemas,
  and storage migrations are unchanged.
- `RuntimeSecretRefSchema` remains patch-compatible with the existing
  `keychain:service:account` shape; stricter keychain segment grammar is
  enforced only at resolver/doctor/runtime secret-use boundaries.
- PathRelation bootstrap templates are not daemon-owned ontology truth;
  daemon defaults provide no built-in templates.
- Zero-day security defaults remain in the runtime policy path
  (`ZERO_DAY_POLICIES_JSON` -> zero-day security layer), not in memory
  object defaults or bootstrap PathRelation seeds.
- Graph health is an advisory doctor diagnostic, not a new public MCP or
  protocol contract.
