Reviewer: general-purpose agent simulating fresh codex receiving P0-4 brief

Scope: same as architect review, plus a "could a fresh codex actually write Phase 1 cards from this?" stress test.

## Blocking (12)

1. **6-section task card template is not actually written down anywhere** with field-by-field examples. Handbook says "see template" — but no template exists. Two parallel codex will produce two incompatible section schemas.
2. **§0 "Charter Authority" content for trivial-copy / adapt-and-port modes is undefined**. Upstream do-what cites a charter; Alaya has none. port-protocol only specs §0 for `requires-redesign`.
3. **`Acceptance Criteria` row format for trivial-copy not specified** (file-level vs package-level, byte-for-byte vs test-pass).
4. **§6 Verification commands need a vitest project that doesn't exist yet** — chicken-and-egg with `packages/<x>/package.json` creation. Phase 1 cards are the first to create those, but §6 references them.
5. **Phase 1 says "~8 cards" but README only lists 7** — what's the 8th? Could be `protocol/src/index.ts` barrel, storage skeleton (db.ts/errors.ts/index.ts), or per-subdir split.
6. **P1-protocol single-card vs multi-card not decided** — source has 124+ files across `src/`, `src/soul/`, `src/events/`. One card vs N cards is a Wave 1 atom-vs-batch decision.
7. **CLI shell command mapping unspecified**: `do-what cli/app` → `alaya doctor/install/attach/status` is the goal, but no card says how to map (delete `app`? rename `cli`? add 4 new commands? — last option pushes to `requires-redesign`).
8. **`docs/v0.1/<phase>-briefs/reports/` directory doesn't exist** and no report template — codex doesn't know report file naming or structure.
9. **Phase 1 missing `protocol/src/event-log.ts` ownership** — it's a root-level protocol file critical for EventLog-first invariant; storage-shared depends on it.
10. **P1-config "and related" is a black hole** — `dynamics-constants-runtime.ts` plus what?
11. **Phase 1 missing `packages/soul/` non-garden leaf files** — signal-handler.ts, worker-safety-*, soul package.json/index.ts skeleton; Phase 2C garden cards will lack soul package shell.
12. **engine-gateway provider/ and tools/ subdirs not addressed** in P1-engine-gateway; trivial-copy may sweep in business logic that violates invariant §5.
