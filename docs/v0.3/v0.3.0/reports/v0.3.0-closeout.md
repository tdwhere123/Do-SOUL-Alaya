## Re-scoping decisions (recorded for the audit trail)

- **Slice 4 strict gate dropped.** The original gate wanted a fresh-install
  capture + recorded raw MCP stdio transcript + offline replay. The daemon
  does not record raw stdio frames in production, and a hand-built
  transcript is exactly the fabricated proof `#BL-038` rejects. The
  live-usage witness keeps the "real host, real conversation" bar while
  being something the operator actually has.
- **`#BL-009` coverage narrowed.** Linux/WSL2 is runtime-verified; macOS
  and Windows adapters are code-reviewed but runtime verification is
  deferred (no maintainer host) — recorded as a bounded deferral in the
  `#BL-009` close note and `maintenance.md`, not a new backlog issue.
