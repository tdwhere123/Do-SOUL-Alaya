CODE COMPLETE in the controller worktree.

## Work Completed

- Added `keychain:<service>:<account>` to the runtime secret-ref schema in
  `@do-soul/alaya-protocol`.
- Added `keychain` to Garden EventLog `secret_ref_kind`.
- Extended `SecretRefReader` with non-optional `readKeychain`.
- Added platform read adapters:

  - Linux: `secret-tool lookup`
  - Windows: PowerShell `Windows.Security.Credentials.PasswordVault`
- Updated config normalization, config audit, Garden credential provenance, and
  secret-resolution error formatters so non-`env:` refs are not mislabeled as
  `file:`.
- Updated protocol semver snapshot for the schema hash changed by
  `secret_ref_kind`.
