# Bench Evidence

One-off audit artifacts that prove the bench harness wiring is real,
not claimed. These are not datasets and not run KPIs — they are
forensic snapshots a reviewer can cross-check against the daemon's
EventLog.

Format is free-form (JSON, markdown, text). Each file should name what
it proves and which commit produced it.

## Contents

- `audit-trail-witness.json` — captured 9-row EventLog chain
  (`soul.signal.emitted` → ... → `soul.proposal.resolved`) for a single
  bench seed driven through the real MCP propose+review chain. Proves
  the bench harness exercises the production governance path instead of
  writing directly through `SqliteMemoryEntryRepo`. Generated by
  `apps/bench-runner/scripts/audit-trail-witness.mjs`.

## Compared to `../datasets/`

- `datasets/` pins **input** checksums for reproducibility (sha256 of
  fetched JSON, source URL, fetched-at timestamp).
- `evidence/` pins **proof of execution path** for behaviors that would
  otherwise require trusting the runner's commit log.
