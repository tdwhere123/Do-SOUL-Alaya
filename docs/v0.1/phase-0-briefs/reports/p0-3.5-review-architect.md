- **B5** `P2-repos-batch-6` lists `event-publisher / runtime-event-normalizer / dynamics-constants` as repos but they are services/constants, not storage repos. Also missing ~14 real remaining repos (deferred-obligation, dirty-state-dossier, strong-ref, tool-execution-record, tool-spec, worker-run, node-instance, cascade-delete, garden-data-ports, drift-lease, bootstrapping-record, extension-descriptor, path-relation, etc.).

- **B7** `Operations (backup, portable, status)` has no owner card; runtime-status uses informal "will land alongside" non-owner.
- **B8** ConversationService (2,133 LOC) has no Alaya consumer (no chat surface). Trivial-copy creates dead code; need explicit scope decision.
